## 作成する参照資料（候補）

### 主要概念・用語集
| 用語 | 説明 |
|------|------|
| AgentCore Runtime | エージェントのデプロイ・実行基盤。コンテナ化されたエージェントをホストする |
| AgentCore Memory | エージェントの会話履歴・長期記憶を永続化するストレージ |
| AgentCore Identity | エージェントが外部サービスへ認証するための OAuth2 ベースの仕組み |
| AgentCore Tools | エージェントが呼び出せる外部ツール・関数の定義・管理 |
| AgentCore Gateway | エージェントへの外部アクセスを管理する API ゲートウェイ |
| MCP | Model Context Protocol。エージェントとツールの標準通信プロトコル |
| Strands Agents | AWS が提供するオープンソースのエージェントフレームワーク |

### 典型的なアーキテクチャパターン（調査後に詳細化）
1. **シンプルな Q&A エージェント**: AgentCore Runtime + Bedrock LLM
2. **記憶を持つ会話エージェント**: Runtime + Memory
3. **ツール呼び出しエージェント**: Runtime + Tools/Gateway
4. **フル統合エージェント**: Runtime + Memory + Identity + Tools
